
## Disney+ Clone

A Clone of Disney+ App that looks just like Disney+ App




## Usage

- https://disneyplusclone-92819.web.app/ 

[OR]

- Run "npm install"
- Add your firebase config details to Server/firebase.js.
- Run "npm start" to run the app
